------------------------Project 1: Rule-Based AI Chatbot------------------------
**Description**
This is an easy-to-make Rule-Based AI Chatbot that uses conditional statements to answer user responses.
**Features**
* Greets Users
* Answers Basic Questions
* Gives Help Information
* Handles Unknown Responses
* Allows users to exit by typing "bye" or "exit"
  
Concepts Used
* Conditional Statements (if-elif-else)
* While Loop
* Handling User Input
* String Manipulation
  
Sample Inputs
* hello
* hi
* how are you?
* what is your name?
* who made you?
* help me!
* bye
  
Learning Outcome
* Learn about rule-based AI
* Learn chatbot programming
* Learn Loops & Conditionals Programming
